# Ticket 1 — Cannot Access Shared Drive

## 📝 Ticket Info

**User:** Sarah Mitchell  
**Issue:** Cannot access shared drive  
**Error:** "Network path not found"  
**Location:** Remote (WFH)

---

## 🔍 Problem Description

User reports mapped network drives are disconnected and inaccessible.  
Other users in the office can access the drive normally.

---

## 🧠 Initial Analysis

- Issue only affects remote user  
- Network connectivity is working  
- Likely VPN / network path issue  

---

## 🛠️ Troubleshooting Steps

1. Verified internet connection  
2. Confirmed issue is not system-wide  
3. Checked mapped drives status  
4. Identified user is working remotely  
5. Determined VPN likely required for access  

---

## ✅ Resolution

User connected to VPN, restoring access to shared drive.
